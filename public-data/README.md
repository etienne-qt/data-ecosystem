# Public Data

This directory contains open-source datasets that are freely available and can be committed to the repo.

## Rules
- Every file must have its source documented in this README
- Only data from public, freely accessible sources belongs here
- Licensed data (Dealroom, PitchBook, etc.) is **never** placed here
