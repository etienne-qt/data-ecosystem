# Controlled Tag Vocabulary

All insights and report summaries use tags from this list only.
To add a new tag, document the rationale here first.

## Topic Tags (12)

| Tag | Maps to | Description |
|-----|---------|-------------|
| `ip-research` | Stage 1 | R&D, patents, university spinoffs, tech transfer |
| `entrepreneurship` | Stage 2 | Company formation, serial founders, diversity |
| `talent` | Stage 3 | Workforce, immigration, compensation, retention |
| `incubation` | Stage 4 | Accelerators, incubators, support programs |
| `funding` | Stage 5 | VC, angel, government financing, rounds |
| `commercialization` | Stage 6 | Revenue growth, international expansion, scaling |
| `exits` | Stage 7 | Acquisitions, IPOs, capital recycling |
| `policy` | Cross-cutting | Government policy, regulation, tax credits |
| `ai-sector` | Sector | AI/ML specific findings |
| `cleantech-sector` | Sector | Clean technology, climate tech |
| `healthtech-sector` | Sector | Health tech, biotech, medtech |
| `macro-trends` | Cross-cutting | Global VC climate, interest rates, macro |

## Geography Tags (5)

| Tag | Description |
|-----|-------------|
| `quebec` | Quebec-specific data or findings |
| `canada` | Canada-wide (includes Quebec in aggregate) |
| `north-america` | US + Canada comparisons |
| `international` | Global or multi-country |
| `{jurisdiction}` | Specific benchmark jurisdiction (e.g., `israel`, `ireland`) — use sparingly |

## Report Type Tags (7)

| Tag | Description |
|-----|-------------|
| `industry-report` | Published by industry association (CVCA, BDC, etc.) |
| `government-report` | Published by government body (StatCan, ISQ, MFQ, etc.) |
| `academic-paper` | Peer-reviewed or university research |
| `ecosystem-ranking` | Global ecosystem benchmarking (Startup Genome, etc.) |
| `survey` | Primary data from surveys (IEQ, founder surveys, etc.) |
| `policy-brief` | Policy analysis or recommendation document |
| `news-synthesis` | Media analysis or curated news summary |

## Credibility Tags (3)

| Tag | When to use |
|-----|-------------|
| `high` | Primary data, known methodology, established publisher |
| `medium` | Secondary synthesis, reasonable methodology, or newer source |
| `low` | Opinion piece, limited methodology, or unverified claims |
